# Invoice AI Platform

An AI-powered invoice processing platform that extracts invoice data using Gemini AI, validates fields, and generates structured Excel reports through a Flask backend and React frontend.

## Tech Stack

- React
- Flask
- Python
- Gemini AI
- Pandas
- OpenPyXL

## Features

- Invoice upload
- AI-based data extraction
- Structured invoice parsing
- Excel report generation
- Modern React dashboard
